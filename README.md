# 安装

使用 pip 安装 etool:

```bash
pip install etool
```

## 网络

### 测试网络速度

```python
from etool import SpeedManager
SpeedManager.network() # 网络测试
SpeedManager.disk() # 硬盘测试
SpeedManager.memory() # 内存测试
SpeedManager.gpu_memory() # GPU测试
```
